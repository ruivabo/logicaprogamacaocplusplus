comandos git 
para sincronizar uma pasta criada do repositório 
git clone <endereço em que está o repositório>.
ele sincroniza o repositório
após a sicronização
clicar na pasta recém sincrozinada 
na barra de endereço apagar tudo e digitar cmd
após o promt de comando estar aberto digitar

git init -> inicia o repositório
git add . -> adiciona todos os arquivos criados 
git commit -m "comentario"-> adiciona um comentário
para ser inserido no repositório
